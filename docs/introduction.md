# Introduction

### While disruptive technologies are becoming the norm, creating a disruptive portfolio gives you an edge that will make you stand out. <a id="While-disruptive-technologies-are-becoming-the-norm,-creating-a-disruptive-portfolio-gives-you-an-edge-that-will-make-you-stand-out."></a>

With a self hosted option, you take complete control of the customization of your portfolio applications, deployment and security. While this can seem like a daunting task, this guide will provide a thorough description of how this is accomplished.

**This tutorial shows you how to deploy and configure various Productivity & Knowledge Base Applications with built-in tools for E-Commerce, CRM, ERP, MRP and more to manage your Portfolio using Docker, deployed on an Ubuntu Server 20.04LTS Hyper-V Virtual Machine.**

## _**Technolgies Involved**_

* Linux Server Administration & CLI
* VirtualBox & Linux Server Virtualisation
* Noip DNS Services
* Networking - Port Forwarding & Firewalls
* Docker & Docker Compose - Containerized Applications & Services
* Nginx Proxy Manager - Proxy Management
* Portainer - Docker Admin via Web UI 
* Cockpit - Server Admin via UI
* Odoo Community Edition - Full Fledged ERP, Accounting, MRP, CRM & Website Designer
* Bookstack - Documentation
* Organizr - Dashboard
* Visual Studio Code - Dockerized Web-Based IDE

The customizable nature of the applications being used means we will be focusing on configuring the stack to function as a Portfolio, with a website interface offering multiple extensabilities. Coupled with multiple options for creating and hosting your own Knowledge Base. All easily managed with web GUI controls.
