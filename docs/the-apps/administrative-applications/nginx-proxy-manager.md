# Nginx Proxy Manager

Access Nginx Proxy Manager at [http://192.168.1.100:81](http://192.168.8.100:81), asssuming that your servers IP address is 192.168.1.100.

81 is the default admin UI port.

The default admin login details are:

> email: admin@example.com
>
> password: changeme

After initial login you will be prompted to change your username, email and password.
